API Documentation

Integrate TikTok video downloading into your application with our free API. No authentication required!

GETYOUR_WEBSITE_URL/api/download

Parameters

Parameter Type Required Description
url string Yes The TikTok video URL

Example Request

GET YOUR_WEBSITE_URL/api/download?url=https://www.tiktok.com/@username/video/1234567890

JavaScript Example

// Fetch TikTok video data
const url = 'https://www.tiktok.com/@username/video/1234567890';
const response = await fetch(`YOUR_WEBSITE_URL/api/download?url=${encodeURIComponent(url)}`);
const data = await response.json();

if (data.status) {
  console.log('Video URL:', data.data.video_url);
  console.log('Creator:', data.data.creator);
  console.log('Likes:', data.data.likes);
}

Python Example

import requests

url = 'https://www.tiktok.com/@username/video/1234567890'
response = requests.get(
    'YOUR_WEBSITE_URL/api/download',
    params={'url': url}
)
data = response.json()

if data['status']:
    print('Video URL:', data['data']['video_url'])
    print('Creator:', data['data']['creator'])
    print('Likes:', data['data']['likes'])
